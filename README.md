# 🏠 Prediksi Harga Rumah (Machine Learning)

Aplikasi ini digunakan untuk memprediksi harga rumah berdasarkan beberapa fitur seperti luas rumah, jumlah kamar, dan kualitas bangunan.

## 🚀 Demo Aplikasi
👉 Klik link berikut untuk mencoba:
https://username-app.hf.space

## 📊 Fitur Input
- GrLivArea (Luas rumah)
- BedroomAbvGr (Jumlah kamar)
- FullBath (Kamar mandi)
- OverallQual (Kualitas rumah)

## ⚙️ Model
Model terbaik yang digunakan:
- XGBoost Regressor

## 📁 Struktur Project
- app.py → aplikasi prediksi
- model_xgb.pkl → model ML
- requirements.txt → dependencies
